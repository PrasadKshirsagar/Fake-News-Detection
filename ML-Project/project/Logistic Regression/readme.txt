--------------------------------------------ABOUT-------------------------------------------------------
This folder contains the code to implement L1 and L2 regularization in Logistic Regression.

'l1norm_log.py' uses either Dataset1 or Dataset2 (hard-coded in the code) and runs Logistic regression using L1 norm. Python glmnet is used for the purpose. 

'l2norm_log.py' uses either Dataset1 or Dataset2 (hard-coded in the code) and runs Logistic regression using L2 norm. 

Compatible with python3 in linux

Dependencies:
->	scipy
->	matplotlib
->	glmnet_python
->	glmnet
->	glmnet_print
->	cvglmnet
->	cvglmnetPlot
->	numpy
->	seaborn

use pip install dependency to install the missing dependency.

Run as 

	python3 l1norm_log.py
	python3 l2norm_log.py
